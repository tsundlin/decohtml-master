Questions You May Have





What is the purpose of this website? At the current stage, the purpose of this website is to help drivers improve their abilty to perceive potential hazards while on the road, through multiple simulation exercises and question exercises.




What makes this website different? This is the only hazard perception website that has simulation exercises. Simulation exercise allows users to interact with a vehicle, manoeuvre the vehicle as needed to prevent hazards. It is interactive, putting users in an actual driving environment, so that the user will be able to perceive hazards while driving in real-life. Lots of other driving perception website does not offer simulation exercise, they only offer questions or videos, which are not interactive, and is not able to immerse the user into the actual scenario. Questions are static, not dynamic, the user is able to focus 100% to identify hazards. Videos are dynamic, but since the user does not need to do anything, simply watch, the user is able to focus 100% to identify hazards. These are not realistic. But with simulation exercises, the user has to control the vehicle while identifying risks, users may not be able to focus on finding hazards completely, as they also need to control the vehicle, which creates a more realistic expericence for the users.




Why is hazard perception test important? A hazard perception test is important because it is a tool to examine driver's ability to identify and react to hazards. Once a hazard has been identified, it is important that the driver is able to react accordingly to increase safety margin. The ability to perceive hazard is a skill that can only be gained overtime. Drivers with less driving experience is prompt to accident due to the disability to react accordingly when a hazard occurred. Therefore, hazard perception test is here to help/assist inexperienced drivers to be exposed with various hazardous environment, allowing them to be able idenfity potential hazards even way before driving on the roads.




Do I need to have a learner licence to use the website? No. Everyone can use the website. Since inexperienced driver or drivers with no experiences (drivers who haven't obtained the learner licence yet) have little ability to percieve hazards, these are the group of people the website targets, to help them improve their abilty to perceive hazards.




What should you do before applying for a learner licence? Before you can apply for a car (C class) learner licence, you need to complete PrepL. In order to be eligible to enrol into PrepL, you have to be at least 15 years and 11 months old. In order to apply for the car (C class) learner licence, you have to be at least 16 years old.




Does this hazard perception test replaces the hazard perception test offered by the Queensland Government? At the current stage, this hazard perception stage is still on an early development stage. Therefore, does not replace the hazard perception test offered by the Queensland Government. However, further trials and testing will be implemented. It plans to replace the current hazard perception test offered by the Queensland Government once the website has become more mature, and stable.
